## Default Permission

Default permissions for the Mint VPN Android plugin: prepare/start/stop the system VPN tunnel and read its status.

#### This default permission set includes the following:

- `allow-prepare-vpn`
- `allow-start-vpn`
- `allow-stop-vpn`
- `allow-vpn-status`
